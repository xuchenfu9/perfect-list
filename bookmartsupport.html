一、常见问题

Q：书签数据会丢失吗？

开启 iCloud 同步后，您的书签数据会自动备份到 iCloud。即使更换设备或重新安装应用，只要登录同一个 Apple ID 并开启 iCloud 同步，数据即可自动恢复。

Q：如何将数据同步到新设备？

请确保两台设备满足以下条件：

  • 在两台设备上登录同一个 Apple ID
  • 在系统「设置」→「Apple ID」→「iCloud」中确认 iCloud 已开启
  • 在「我的书签」应用的设置中开启 iCloud 同步
  • 等待片刻，数据将自动同步
Q：Face ID / Touch ID 解锁失败怎么办？

当 Face ID 或 Touch ID 不可用时，系统会自动切换为设备密码验证。如果仍然无法解锁，请检查：

  • 确认设备已设置 Face ID / Touch ID 或设备密码
  • 前往系统「设置」→「Face ID 与密码」确认功能正常
  • 尝试重启应用后再次解锁
Q：免费版有什么限制？

免费用户最多可保存 20 个书签和 3 个私密书签。升级为会员后可解锁无限书签收藏，无任何数量限制。

Q：如何恢复已购买的会员？

进入应用「设置」页面，在会员状态区域点击「恢复购买」按钮即可。请确保使用的是购买时的同一个 Apple ID。

Q：支持哪些语言？

目前支持以下 8 种语言：简体中文、繁體中文、English、日本語、한국어、ไทย、Italiano、Deutsch。您可以在设置中切换语言，也可以选择跟随系统语言。

Q：如何更换默认浏览器？

进入应用「设置」→「浏览器设置」，可选择 Safari、Chrome、Firefox 或 Edge 作为默认打开方式。对于私密分类中的书签，部分浏览器还支持自动使用隐私模式打开。

二、数据备份与恢复

「我的书签」支持将所有书签数据导出为 JSON 文件，方便您手动备份或迁移数据。

导出数据

  • 打开应用，进入「设置」页面
  • 在「备份与恢复」区域点击「导出书签」
  • 选择保存位置，文件将以 JSON 格式导出

导入数据

  • 打开应用，进入「设置」页面
  • 在「备份与恢复」区域点击「导入书签」
  • 选择之前导出的 JSON 文件
  • 系统会自动去重，已存在的书签不会重复导入
💡 提示：建议定期导出备份，以防数据意外丢失。导出文件包含所有分类和书签信息。

三、iCloud 同步

本应用通过 Apple CloudKit 实现 iCloud 同步，您的数据在传输过程中均经过加密处理。

同步异常排查

  • 确认设备已连接网络（Wi-Fi 或蜂窝数据）
  • 确认 Apple ID 已登录且 iCloud 服务正常
  • 在应用设置中点击「检查 iCloud 状态」查看连接情况
  • 如显示「iCloud 不可用」，请前往系统设置检查 iCloud 账户状态
  • 尝试关闭再重新开启 iCloud 同步开关
注意：关闭 iCloud 同步后，数据将仅保存在当前设备本地。卸载应用会删除本地数据。如需彻底清除 iCloud 中的同步数据，请前往系统「设置」→「Apple ID」→「iCloud」→「管理存储空间」中手动删除。

  • 如遇购买问题，请确认 Apple ID 的支付方式有效，或联系 Apple 支持

六、故障排除

应用闪退或无响应

  • 确保应用已更新至最新版本
  • 尝试强制关闭应用后重新打开
  • 重启设备后再次尝试
  • 如问题持续，请通过邮件联系我们并附上设备型号和系统版本

书签无法打开

  • 检查网址是否正确（应以 http:// 或 https:// 开头）
  • 确认所选浏览器已安装在设备上
  • 尝试切换到 Safari 浏览器打开
